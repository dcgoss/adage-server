Genes

ADAGE model is built with genes in Pseudomonas aeruginosa PAO1 strain. Information about genes are obtained from NCBI.

Samples

We include samples measured on the Affymetrix Pseudomonas aeruginosa Array "Pae_G1a" (labeled as A-AFFY-30 in ArrayExpress database).

Experiments

Experiments are obtained from the ArrayExpress database. Each experiment has a “E-XXXX-number” accession number defined by ArrayExpress.

Expression compendium?

The webserver is currently using P.a. expression compendium. It was built on July 31 2015 and contains 125 datasets with 1051 individual samples.

Nodes

Nodes are building units in an ADAGE model. Each node is defined by a weight vector storing weight value of each gene in that node.

Signatures

Signatures are functional gene sets extracted by an ADAGE model. A signature contains genes contributing extremely positive high or negative high weights to a specific node. One node results in two signatures on positive and negative sides. The positive and negative signatures are treated independently.
